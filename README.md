# react-native-popupmenu
Provide a programmatical api for showing popup menu
## Installation

```sh
npm install react-native-popupmenu
```

## Usage

```js
import { PopupmenuView } from "react-native-popupmenu";

// ...

<PopupmenuView color="tomato" />
```
